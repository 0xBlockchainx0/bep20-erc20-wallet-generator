# ethereum-wallet-generator

> Simple Ethereum Wallet Generator Tool

## How to start
### Install dependencies
```bash
# via yarn
$ yarn
# or via npm
$ npm install
```

### Run script
```bash
# random wallet address
$ node index.js

# address with desired prefix
$ node index.js aaa
```

## Features
- No Internet connection needed
- Generate address with desired prefix
